# SensorLympics

Sensor based Olympic Games

## How to build
  - Generate APi key from: https://openweathermap.org/api

## Games
  - Single player games:
    - Magneto Game
    - Pressure Game
    - Ball Game

  - Multiplayer game:
    - Tic Tac Toe
